# 2025-05-17 工作日志

## Bug 修复：人物卡片删除后 Drawer 不关闭

### 问题
删除人物卡片后，`CharacterDrawer`（侧边抽屉）没有关闭，仍然显示已删除人物的旧数据。

### 原因
`CharacterPage.tsx` 的 `handleDelete` 只执行了 `setDeleteTarget(null)` 关闭确认弹窗，没有重置 `drawerOpen` 和 `drawerChar` 状态。

### 修复
在 `handleDelete` 中增加两行：
```tsx
setDrawerOpen(false);
setDrawerChar(null);
```
