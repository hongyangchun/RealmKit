# 2026-05-21 工作日志

## Remotion 视频渲染 — 世界圣典宣传片

### 任务背景
WorkBuddy 平台当前会话的视频生成工具（HY-Video、YT-Video）无法在 Agent 工具链中调用，改用 Remotion 方案以纯代码方式生成产品宣传视频。

### 产出
- **视频文件**：`/Users/hongyangchun/Codebase/RealmKit/video-promo/out/zzworld-promo.mp4`
- **规格**：1344×768，30fps，H.264，15 秒，10.8 MB
- **内容结构**：
  - Scene 1（0–3s）：开场，第一张关键帧图 + 品牌标语「为你的世界立典」，Ken Burns 效果
  - Scene 2（3–7s）：主标题「世界圣典 / ZZWorld Chronicle」弹入动画 + 金色装饰线
  - Scene 3（7–11s）：三大特性展示（🗺️ 手绘地图 / 👥 人物关系网 / 📜 历史编年），依次上浮出现
  - Scene 4（11–15s）：CTA 结尾，金色粒子漂浮 + 呼吸光晕 + 「开始创作你的世界」

### 技术栈
- **Remotion 4.0.464** + React 18 + TypeScript
- 项目目录：`/Users/hongyangchun/Codebase/RealmKit/video-promo/`
- 渲染命令：`./node_modules/.bin/remotion render src/index.ts PromoVideo out/zzworld-promo.mp4`
- 渲染用时：约 30 秒（450 帧，5x 并发）
- 图片资源来自 `generated-images/` 目录（珀西生成的 4 张关键帧）

### 注意事项
- 必须 `unset NODE_OPTIONS` 再执行 node/npm/remotion 命令，否则报 `--use-system-ca is not allowed in NODE_OPTIONS`
- Remotion 会自动下载 Chrome Headless Shell（首次 ~93MB，后续缓存）
- 字体（LXGW WenKai TC）在渲染环境中可能降级到系统衬线体，视觉效果略有差异
